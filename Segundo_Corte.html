Durante el curso de Lenguajes de Programación, se abordaron conceptos fundamentales de Python, incluyendo el uso de funciones incorporadas como `map`, `reduce`, y `filter`. Estas funciones son herramientas poderosas que permiten aplicar funciones sobre otras funciones o datos de manera eficiente:

  • Función `map`:
    `map` es utilizada para aplicar una función a todos los elementos de una lista iterable y devuelve un nuevo iterable con los resultados.
  • Función `filter`:
    `filter` se emplea para filtrar elementos de una secuencia según una función condicional y devuelve aquellos elementos que cumplen con esa condición.
  • Función `reduce`:
    `reduce` permite aplicar una función a pares de elementos de una secuencia para reducirlos a un solo valor acumulado.
